# 🩺 AI Healthcare Assistant

## Project Overview

AI Healthcare Assistant is a Python-based web application that predicts possible diseases from user symptoms and suggests general home remedies using the IBM Granite Large Language Model (LLM).

> **Disclaimer:** This project is developed for educational purposes only. It does not provide medical advice or replace consultation with a qualified healthcare professional.

---

## Features

* Predict possible diseases from symptoms
* Suggest general home remedies
* Simple and user-friendly interface
* Powered by IBM Granite AI

---

## Technologies Used

* Python
* IBM Granite LLM
* Hugging Face Transformers
* Hugging Face Hub
* Gradio

---

## Project Workflow

User enters symptoms → AI model analyzes the symptoms → Disease prediction is generated → General home remedies are suggested.
